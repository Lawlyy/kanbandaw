# Tablero Kanban - DAW06

## Descripción

Este proyecto consiste en una aplicación web tipo Kanban hecha con HTML, CSS, Tailwind y JavaScript.

La aplicación permite gestionar tareas organizadas en tres columnas:

- Por hacer
- En curso
- Hecho

También permite crear, editar, eliminar y mover tareas entre columnas. Además, incluye filtros, búsqueda, estadísticas y guardado de datos usando localStorage para que las tareas no se pierdan al recargar la página.

## Funcionalidades

- Crear tareas con título, descripción, prioridad y fecha límite.
- Editar tareas existentes.
- Eliminar tareas.
- Cambiar el estado de las tareas.
- Filtrar tareas por estado.
- Filtrar tareas por prioridad.
- Buscar tareas por texto.
- Mostrar estadísticas básicas.
- Guardar las tareas automáticamente en localStorage.

---

## Guía rápida de uso

### Crear una tarea

Rellenar el formulario y pulsar el botón **Añadir tarea**.

### Editar una tarea

Pulsar el botón **Editar** de una tarea. Los datos aparecerán en el formulario y se podrán modificar.

### Eliminar una tarea

Pulsar el botón **Eliminar**. Aparecerá una confirmación antes de borrarla.

### Cambiar el estado

Usar el desplegable de estado para mover la tarea entre las columnas.

### Filtrar y buscar

Usar el buscador o los filtros para encontrar tareas más fácilmente.
